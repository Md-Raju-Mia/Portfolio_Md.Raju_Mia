Project Overview

A blog page website with several catagories of blogs like "Trend blogs", "Latest Stories", "Article", "Case Study", "News". Each block is clickable for details information about the blogs. A person can also shows the related blocks he likes to read. A person also can 1. Share blogs on facebook, linkedIn, Whatsapp via link. 2. Visitors can make inquiries using the contact form. 3. Website owner can create blog post, delete blog post and also be able to update.
